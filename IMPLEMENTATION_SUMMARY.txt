================================================================================
                            HOW IT WORKS
================================================================================

USER FLOW:
1. User browses shop (/shop)
2. Clicks "Add to Cart" on any product
3. Form posts product data to CartController@add
4. Controller validates and stores in session
5. User redirected to cart page (/cart)
6. Cart displays all items with order summary
7. User can modify quantities or remove items
8. Totals calculate automatically

DATA STORAGE:
- Session-based (no login required)
- Survives page navigation
- Each product has unique key: md5(title + format)
- Duplicate products increment quantity automatically

CALCULATIONS:
- Subtotal = SUM(price × quantity)
- Tax = Subtotal × 8%
- Shipping = FREE
- Total = Subtotal + Tax
